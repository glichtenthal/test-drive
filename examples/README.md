# Test Drive Examples

These examples show how Test Drive stays horizontal by routing the same core question through different evidence types:

> What kind of evidence would make this idea more or less trustworthy?

## Examples

- [Positioning Test](positioning-test.md) - human reaction and artifact performance evidence for a message before publishing.
- [Data Analysis Test](data-analysis-test.md) - behavioral/data evidence for a retention claim before jumping to regression.
- [Skill Evaluation Test](skill-eval-test.md) - artifact performance and reasoning evidence for a skill before release.

## How To Read Them

Each example shows the input, evidence type, smallest credible test, artifact needed, and signals that would support or weaken the idea. Use them as patterns, not rigid templates.
